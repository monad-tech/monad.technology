# Monad Technology Website

This is the [official website for Monad Technologies LLC](https://monad.technology),
a consulting firm specializing in distributed systems and cryptography protocols.

## Tech Stack

- **Svelte 5**: The latest version of the Svelte framework.
- **Vite**: Build tool.
- **Vanilla CSS**: Custom premium styling.

## Development

To start the development server:

```bash
npm run dev
```

## Building

To create a production build:

```bash
npm run build
```
